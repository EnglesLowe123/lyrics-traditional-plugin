# MoeKoeMusic 簡繁轉換工具

適用於歌詞的簡繁轉換

## 安裝方法

### 方法一：透過插件市場安装

在 MoeKoeMusic 設定 → 插件管理找到『歌词简繁转换』安裝。

### 方法二：手動安裝

1. 将插件資料夹複製到 MoeKoeMusic 的插件目錄：
   - **Windows**: `%APPDATA%/MoeKoeMusic/extensions/`
   - **macOS**: `~/Library/Application Support/MoeKoeMusic/extensions/`
   - **Linux**: `~/.config/MoeKoeMusic/extensions/`
2. 重新啟動 MoeKoeMusic
